JSON serializer/deserializer

The name of the java bean has to be the same as declared in the JSON object. If the name is different we use the @SerializedName annotation. Eg: @SerializedName("my name")


Customization of the json's array size is also possible using the @ItemType annotation. Eg: @ItemType(size=10). This helps when we need to restrict the number of array elements to be deserialized.
